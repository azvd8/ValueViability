# Changelog

## v1.0.0 — 2026-05-15

Initial reproducibility package for the Value-Viability Framework illustrative application.

Included:

- R script for synthetic data generation;
- BCV summaries;
- Value and Viability composite score calculation;
- 2x2 median-based classification;
- 3x3 tertile-based classification;
- vector figure generation in PDF and SVG.
