# Data folder

This folder is reserved for data files associated with the reproducible illustration.

In the current version, the datasets are generated directly by the R script:

- `bcv_panel_responses_synthetic.csv`
- `bcv_summary_synthetic.csv`
- `project_classification_results.csv`

These files are synthetic and used only for methodological illustration.

After running `R/value_viability_framework.R`, the generated CSV files may be saved in this folder if the repository maintainers decide to include the generated data outputs.
