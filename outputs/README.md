# Outputs folder

This folder is reserved for generated figures and other reproducibility outputs.

The R script generates the following vector graphics:

- `BCV.pdf`
- `BCV.svg`
- `2x2_matrix.pdf`
- `2x2_matrix.svg`
- `3x3_matrix.pdf`
- `3x3_matrix.svg`

Recommended practice:

1. Run the R script.
2. Check whether the graphics were generated correctly.
3. Move or copy the final output files to this folder.
4. Commit only the final figures that correspond to the version cited in the article.
